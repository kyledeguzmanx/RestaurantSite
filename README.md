# CSUN BURGER
A mock-site for a burger chain called CSUN Grill. It includes an image carousel, navigation bar, navigation bar w/ dropdown, and CSS transitions

## Technology
Adobe Photoshop 2022  
Bootstrap  
Canva  
CSS Transitions  
EmailJS  
Grid + Flexbox  
HTML  
Javascript    
VueJS
